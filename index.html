Mis proyectos

En esta sección se encuentran algunos de mis proyectos, cuyos códigos fuente se encuentran publicados en mi repositorio de Github. Cada proyecto tiene una temática diferente puesto que pertenecen a trabajos académicos y colaboraciones con equipos externos.

Plataforma Web de Reservas de Hoteles y Tours

Plataforma con módulos de usuarios, proveedores, reseñas y planificación de viajes. Backend implementado en Laravel, APIs REST, autenticación y despliegue completo en Render.

Angular Laravel Python Docker Render API REST Git/GitHub

AGMAC – App móvil de control de medicación

Aplicación móvil desarrollada en Kotlin con Jetpack Compose para gestionar horarios de medicación en adultos mayores. Incluye lectura, edición y filtrado de datos desde archivos JSON, notificaciones programadas y flujo completo de registro de medicamentos.

Kotlin Jetpack Compose JSON Android

ScrapeFlow — Web Scraping Paralelizado

Herramienta paralelizada para comparar productos en múltiples tiendas online. Se implementó scrapers optimizados en Python, manejo de lazy loading, detección de estructuras HTML heterogéneas y estrategias anti-bloqueo para mejorar el rendimiento.

Python Selenium BeautifulSoup C Automatización Optimización Scraping

NovaInfor — Chatbot de Orientación Profesional

Chatbot desplegado en Telegram para guiar a estudiantes de Ingeniería Informática en exploración de especialidades y oportunidades laborales. Se integro la API de OpenAI, datos estructurados y despliegue en Google Cloud con disponibilidad continua.

Python Telegram Bot API OpenAI API Google Cloud VM Tmux Data Cleaning

Página Web + PHP + SQL

Este proyecto consiste en una página web enlazada a una base de datos local a traves de PHP. Desde la página se puede visualizar, editar y eliminar la información de la base de datos.

PHP SQL HTML/CSS

Web con Flask

En este proyecto se utilizo el framework Flask para crear una página web de un restaurante. Esta cuenta con función de Login, Inicio de Sesión y el uso de formularios. La página muestra una diferente estructura para el dueño y para los consumidores.

Python Flask HTML/CSS Javascript

Habilidades Interpersonales

Reflexiones sobre las habilidades blandas que he fortalecido y cómo las aplico en mi vida académica y personal.

Trabajo en equipo

Durante el desarrollo de la Plataforma Web de Reservas de Hoteles y Tours, coordiné tareas con el equipo de frontend y backend usando Git y metodologías ágiles. Esta experiencia consolidó mi habilidad para colaborar, dividir responsabilidades y cumplir entregables en equipo.

Comunicación efectiva

En el proyecto DataInsight CSMC, desarrollé un dashboard para un centro de salud mental, donde tuve que entender las necesidades del cliente y traducirlas en indicadores visuales claros. Esto fortaleció mi capacidad de comunicar ideas técnicas de forma efectiva a personas sin formación técnica.

Adaptabilidad y aprendizaje rápido

Todos los proyectos que he desarrollado han implicado tecnologías nuevas para mí. He tenido que aprender rápidamente durante el proceso cada herramienta y lenguaje para poder completar los entregables, demostrando mi capacidad de adaptarme y adquirir conocimientos bajo presión.

Creatividad y resolución de problemas

En el proyecto NovaInfor, inicialmente queríamos implementar el chatbot en WhatsApp, pero nos encontramos con restricciones en la API. Buscamos otras alternativas y logramos implementar exitosamente el chatbot en Telegram, asegurando la funcionalidad del proyecto.
